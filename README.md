# 4xxbypass

A tool that automates a number of well-known 403/401 bypassing techniques. 

# Usage

`echo "https://target.com/this/is/a/403/page" | 4xxbypass`

`cat urls.txt | 4xxbypass`

# Dependencies

httpx (https://github.com/projectdiscovery/httpx)
